# List merged branches from git

## Before start
This app requires git and grep installed on your machine

## Installation
You can download releases for apple from  [releases](https://github.com/semiherdogan/merged-branches/releases) page.
Or you can download source code and compile on your own.

## Usage
`branch-merged [branchName] [user]`

* To see list of branches that merged into main branch and not deleted use:
```
branch-merged main
```
